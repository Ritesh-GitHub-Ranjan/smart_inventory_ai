# API Reference

## Core Endpoints

### `/api/inventory`
- **GET**: Retrieve current inventory levels
- **POST**: Update inventory items

### `/api/forecast`
- **GET**: Get demand forecasts
- **POST**: Trigger new forecast generation

## Agent Endpoints

### `/api/agents/reorder`
- **POST**: Trigger reorder calculations

### `/api/agents/pricing`
- **POST**: Run pricing optimization
